# Memoteca
## Esse é o memoteca, um mural de pensamentos onde você pode exibir seus melhores pensamentos, com funcionalidades de um CRUD completo, podendo adicionar, remover e editar um pensamento.
## Tecnologias usadas:
- HTML
- CSS
- Typescript
- Angular
- Json Server
